# Quotify

Quotify is an app written in Go for managing quotes. It utilizes a JSON file-based
database and simplifies the tasks of storing, retrieving, searching and sanitizing
favorite quotes..

## Goals
- Ease of storing favorite quotes
- Profaniy checker
